# IGCSE 0478 & 9618 Pseudocode Interpreter

An interactive, web-based pseudocode interpreter designed for **Cambridge IGCSE Computer Science (0478)** and **AS & A Level Computer Science (9618)** students.

This project helps students **practice, understand, and test exam-style pseudocode** by executing it step-by-step in a browser-based environment.
